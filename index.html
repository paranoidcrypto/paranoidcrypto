PARANOIDCRYPTO

Codifica tus mensajes en secuencias de ADN (A, T, G, C). Transforma texto plano en una hebra genética indescifrable sin la clave correcta. Porque hasta el silencio puede tener forma de vida.

[ TERMINAL // DNA_ENCODER ] SYSTEM ONLINE
BYTES ORIG: 0 BYTES COMP: 0 NUCLEOTIDES: 0 RATIO: 0x

HOW IT WORKS

STEP_01

BWT + MTF + Deflate

El texto se transforma con Burrows-Wheeler y Move-to-Front, luego se comprime con Deflate para ocultar patrones y reducir tamaño.

STEP_02

XOR with Key

Si defines una clave, se aplica un XOR cifrado usando SHA-256 como stream.

STEP_03

Nucleotide Mapping

Cada 2 bits se traducen a una base usando el diccionario seleccionado.

STEP_04

Resulting Strand

Obtienes una secuencia de ADN que puede revertirse con la misma clave y diccionario.
