PARANOIDCRYPTO

Codifica tus mensajes en secuencias de ADN (A, T, G, C). Transforma texto plano en una hebra genética indescifrable américa la clave correcta. Porque hasta el silencio puede tener forma de vida.

HOW IT WORKS

STEP_01

UTF-8 → Binary

El texto se convierte a bytes UTF-8 y luego a una cadena de bits.

STEP_02

XOR with Key

Si defines una clave, se aplica un XOR cifrado usando SHA-256 como stream.

STEP_03

Nucleotide Mapping

Cada 2 bits se traducen a una base: 00→A, 01→T, 10→G, 11→C.

STEP_04

Resulting Strand

Obtienes una secuencia de ADN que puede revertirse con la misma clave.
